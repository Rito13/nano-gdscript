# nano-gdscript
GDScript Syntax Highlighting in GNU Nano.
Contributions are welcome!

# Installation
1. Admit that `install` is an executable
```sh
chmod +x install
```
2. Start the installation. On android use `-a` option. 
```sh
./install
```
3. Test it
```sh
nano test.gd
```
